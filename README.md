# 📈 Zero-Dollar Trading Dashboard

A single-file news trading simulator. No backend, no API costs, free forever.

**[🔗 Live Demo](https://yourusername.github.io/zero-dollar-trader)**

---

## What It Does

- 📰 Pulls live RSS feeds (WSJ, Google News, Reddit) via free proxies
- 🏷️ Tags headlines (Trump, Fed, econ, geo, insider) in the browser
- 🤖 AI analysis (Gemini 2.0 Flash-Lite) — optional, paste your own free key
- 💰 Paper trading with $1,000 virtual cash (saved in localStorage)
- 🚀 Runs 100% in the browser — zero backend, zero cost

---

1. Download `index.html`
2. Open in browser
3. (Optional) Add [free Gemini API key](https://aistudio.google.com/app/apikey) for AI analysis
